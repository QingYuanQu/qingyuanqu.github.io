工信部职业技术认证资格考试

工信部职业技术认证是由工业和信息化部教育与考试中心（工业和信息化部直属事业单位，以下简称工信部教育与考试中心）统筹组织。主要围绕信息技术应用、计算机软件、网络安全、装备制造、移动通信技术等领域为重点，整合行业教育资源，联合行业领军企业，组织行业专家团队、科研专家团队共同研究产业政策，制定行业人才培养和评价标准，进行人才培养和选拨工作。

近几年来大数据产业发展规模非常迅速，预计2020整个产业规模将达到1000亿元。但由于产业发展起步晚，截止当前，在大数据领域的人才评价体系寥寥无几，极大影响了大数据相关人才关于大数据应用技术能力和水平的鉴定，同时也影响了用人单位对于所招聘大数据技术人才的能力参考。工信部教育与考试中心大数据技术认证是目前国内最具官方代表性、最具权威性、最具公信力和受行业企业普遍认可的行业人才评价机构。

针对大数据主流技术，相关大数据职业技术认证共设四个方向，分别是大数据应用技术（Hadoop)、大数据应用技术（Spark)、算法工程师、机器学习训练。每个方向技术认证均设初、中、高三个认证等级。

证书等级 报考条件
初级证书 中职、中专、高职、高专学历在读学生或社会在职人员。
中级证书 通过初级认证的中职、中专、高职、高专学历的在读学生或社会在职人员；
本科及以上学历在读学生或社会在职人。
高级证书 通过中级认证资格的中职、中专学历在读学生或社会在职人员；
通过初级认证资格的高职、高专学历在读学生或社会在职人员；

id 选择器

选择器可以是 id 选择器，就是为元素指定一个唯一 id，为这个 id 的元素指定样式时， 样式仅仅被应用到这一个元素，不可能有更多的元素。下面我们用外部样式表来演示。

类选择器

选择器可以是类选择器，就是为多个元素指定一个类名，为这个类指定样式时， 样式将被应用到这个类的所有元素。下面我们还是用外部样式表来演示。

（1）将 css.html 中的“Line 1”、“Line 3”、“Part Tow”三行指定 class 为“blue-text”

（2）在外部样式表文件 bookstore.css 中，指定类 blue-text 的样式为颜色 blue。

完成后，在浏览器刷新页面，看到有四行是蓝色的，其中三行是段落，一行是三级标题。
